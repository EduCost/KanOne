
# KanOne - Kanban App for Android

KanOne is a offline kanban app, designed to help you organize tasks and manage projects

## ✨ Features

- **Easy to use**
- **Create, Edit, and Delete Tasks**
- **Create labels**
- **Attach images** 
- **Drag & Drop**
- **Zoom in & Zoom out**
- **Material You**
- **Offline**

<br>

## 🛠️ Tech Stack
- Jetpack Compose
- Dagger Hilt
- RoomDB
- Coil
- Preferences DataStore
